# 喵了个咪 · 页面开发需求文档

> 基于猫咪三区循环系统设计，版本 v2.2 · 2026-05-09 三区架构 + 亲密度系统

---

## 一、页面结构总览

```
底部导航栏（4个Tab）
├── 首页        主页大厅，出战区快览 + 通知
├── 消消乐      消除游戏，产出鱼干
├── 家园        备战区/家园/包裹区 三区管理
└── 我的        个人中心，称号/里程碑/纪念册
```

---

## 二、全局组件

### 顶部货币栏（所有页面常驻）

```
[鱼干图标] 128    [猫铃铛图标] 3
```

- 鱼干：日常高频资源，喂猫消耗
- 猫铃铛：稀缺资源，抽卡专用，整数显示
- 点击任一货币 → 弹出来源说明浮层

### 通知横幅（首页 + 猫咪页）

优先级从高到低，同时只显示最高优先级一条：

| 优先级 | 触发条件 | 文案 |
|--------|---------|------|
| 1 | 家园有猫正在告别中（弃养24h窗口） | "橘猫 #234 正在收拾行李，还有X小时" |
| 2 | 包裹区猫咪剩余 ≤ 7 天 | "包裹区有1只猫还有7天就会离开" |
| 3 | 家园有猫亲密度 ≤ 20 | "橘猫 #234 亲密度很低，快去喂它吧" |
| 4 | 包裹区有猫待领养 | "包裹区有2只猫等你领养" |
| 5 | 出战席阵容不完整 | "出战席还有空位，去调整阵容吧" |

点击横幅 → 跳转到对应猫咪详情页

## 三、首页（Tab 1）

### 3.1 布局结构

```
┌─────────────────────────────┐
│  [头像] 昵称  称号标签        │  ← 顶部用户信息
│  [通知横幅]                  │  ← 条件显示
├─────────────────────────────┤
│ 出战席                       │
│  [1号·主动] [2号·主动] [3号·主动] │
│                             │
│ 家园                         │
│  [猫1] [猫2] [猫3]           │
│                             │
├─────────────────────────────┤
│  [签到]  今日任务进度        │  ← 每日区域
├─────────────────────────────┤
│  消消乐入口                  │  ← 快捷入口
│  “今日已获得 12 条鱼干”      │
└─────────────────────────────┘
```

### 3.2 出战席预览区

- 固定3个槽位（不扩容），暗金大理石圆台视觉
- 点击调整阵容，进入“备战区”
- 前 3 格标注「主动」标签，对应消消乐底部 3 个技能按钮
- 每个已占用槽位显示猫咪卡面（含稀有度边框颜色）
- 空槽显示 `+` 虚线框，点击 → 跳转猫咪页出战席 Tab
- 点击任意猫咪卡 → 跳转猫咪详情页

### 3.3 家园预览区

- 展示家园中第 1、2、3位猫咪
- 已占用槽位显示猫咪卡面 + 亲密度进度条：❤️ 85/100
- 空槽显示 `+` 虚线框，点击 → 跳转家园 Tab

**亲密度视觉规则：**

| 亲密度区间 | 进度条颜色 | 附加动效 |
|-----------|---------|---------|
| 80-100 | 粉色满条 | 无 |
| 50-79 | 粉色半条 | 无 |
| 20-49 | 灰色半条 | 无 |
| 0-19 | 灰色空条 | 轻微抖动 |

### 3.3 签到区域

- 今日是否已签到状态
- 签到奖励：鱼干 × 5（连续7天：第7天额外给猫铃铛 × 1）
- 点击签到 → 动画播放 → 货币增加

### 3.4 消消乐快捷入口

- 显示今日已通过消消乐获得的鱼干数
- 按钮文案：`去打消消乐` / `今日继续`
- 点击 → 跳转消消乐页

## 四、猫咪页（Tab 3）

### 4.1 布局结构

```
┌─────────────────────────────┐
│  [备战区] [家园] [包裹区]     │  ← 三区 Tab 切换
├─────────────────────────────┤
│                             │
│   （备战区 / 家园 / 包裹区）   │
│                             │
└─────────────────────────────┘
```

### 4.2 备战区 Tab
备战区分为出战席和观战区，为上下结构分布。
上面3个槽位（不扩容），暗金大理石圆形站台风格。上面3格为主动技能位。
下面观战区，为被动技能位，游戏全局生效。初始为0个槽位，通过里程碑解锁3格.

**出战席槽位卡片（每张卡）：**

```
┌──────────────────┐
│  [猫咪图片]       │
│  橘猫             │
│  #0234  N        │
│  🎯 主动技能      │  ← 1-3号位
│  挑出来           │
│  [撤回家园]       │  ← 操作按钮
└──────────────────┘
```
**备战区槽位卡片（每张卡）：**
```
┌──────────────────┐
│  [猫咪图片]       │
│  幸运猫           │
│  #001  SSR       │
│  🎯 被动技能       │  ← 4-6号位
│  福星             │
│  [撤回家园]        │  ← 操作按钮
└──────────────────┘
```

- 1-3 号位标注「主动」标签 + 技能名，对应消消乐底部按钮
- 4-6 号位标注「被动」标签 + 技能名，游戏全局生效，对应消消乐顶部消息
- 从家园拖拽猫咪上场，或点击空槽 → 弹出家园猫咪选择列表
- 点击「撤回家园」→ 猫咪回到家园，槽位释放
- 撤回/上场零成本、零等待

**阵容校验：**
- 黑猫与三花猫不能同时出战，冲突时系统提示「这两只猫性格不合，不能同时出战」

**出战席全空状态：**

```
尚未配置出战阵容
从家园选择猫咪上场吧
[去家园]
```

### 4.3 家园 Tab

家园容量初始10格。猫咪永生不流浪，通过喂食提升亲密度。家园后续扩容，使用鱼干扩容，每格150鱼干。

**槽位卡片（每张卡）：**

```
┌──────────────────┐
│  [猫咪图片]       │
│  星空猫           │
│  #0023  SSR      │
│  ❤️ 85/100       │  ← 亲密度
│  🎯 被动: 观星    │  ← 被动技能
│  [喂食 -1鱼干]   │  ← 操作按钮
│  [批量补满]      │  ← 快捷按钮
│  [上场]          │  ← 移至出战席
│  [送走]          │  ← 送去流浪
└──────────────────┘
```

- 喂食按钮点击：
  - 鱼干 ≥ 1：扣除1条，亲密度+5，播放喂食动画
  - 鱼干 = 0：按钮置灰，提示"鱼干不足，去打消消乐获得"
- 批量补满按钮：
  - 扣除所需鱼干（(100-当前亲密度)/5 条），直接补满亲密度
- 上场按钮：选择出战席空位，猫咪移入出战席
- 送走按钮：触发告别仪式（24h窗口）

**空槽：**

```
┌──────────────────┐
│                  │
│    + 领养新猫     │
│  包裹区有2只等你   │  ← 有猫时显示数量
└──────────────────┘
```

点击 → 跳转包裹区

**里程碑解锁提示（第4/5/6格）：**

若槽位未解锁，显示灰色锁定状态：

```
┌──────────────────┐
│     🔒           │
│  领养10只后解锁   │
│  当前 3/10       │
└──────────────────┘
```

### 4.4 包裹区 Tab

**列表项（每只猫）：**

```
┌─────────────────────────────────┐
│ [猫图] 橘猫  #0234  N           │
│        性格：傲娇 · 吃货         │
│        ⏰ 还有 23 天会离开       │
│                    [领养]        │
└─────────────────────────────────┘
```

**点击领养按钮逻辑：**

- 家园有空槽 → 确认弹窗：
  ```
  "正式领养这只橘猫 #0234？
   它将在这里永远陪伴你，喂食提升亲密度。"
  [取消]  [领养它]
  ```
  确认后：播放领养动画，猫移入家园，亲密度=0

- 家园已满 → 提示弹窗：
  ```
  "家园已满（10/10）
   需要先送走一只猫才能领养新成员。"
  [去家园]  [取消]
  ```

**空状态：**

```
包裹区空空如也
去抽卡带回一只猫吧
[去抽卡]
```

## 九、核心交互流程

### 流程 A：每日循环

```
打开App
  → 查看首页通知横幅（是否有猫亲密度低/包裹区到期/阵容空缺）
  → 点击签到（获得鱼干）
  → 检查出战席阵容（拖拽调整）
  → 打一局消消乐（获得鱼干）
  → 去家园喂猫提升亲密度（消耗鱼干）
  → 查看包裹区（决定是否领养新成员）
  → 关闭App
```

### 流程 B：抽卡到领养

```
用猫铃铛抽卡
  → 猫进入包裹区（30天计时开始）
  → 收到通知（还有7天到期）
  → 打开包裹区
  → 家园有空槽 → 点击领养 → 确认 → 猫进家园（亲密度=0）
  → 家园已满 → 决定送走一只 → 腾出槽位 → 领养
```

### 流程 C：猫咪去流浪（仅由主动弃养触发）

```
玩家主动送走 → 告别仪式开始（24小时）
  → 首页通知横幅显示（最高优先级）
  → 玩家24小时内鱼干挽留 → 猫回来（亲密度不变）
  OR
  → 24小时内未挽留 → 猫流浪
  → 序列号进入60天冷却
  → 槽位释放
  → 猫进入纪念册
```

### 流程 D：里程碑解锁第4格

```
历史领养达到10只
  → 系统弹出通知：「观赛区第4格已解锁！」
  → 家园新增一个空槽
```

### 流程 E：流浪猫救回

```
发现流浪猫（悬赏/活动）
  → 救回
  → 猫直接进入家园（亲密度=0）
  → 恢复成长档案记录
```

### 流程 F：出战阵容配置

```
打开猫咪页 → 出战席 Tab
  → 查看当前阵容（1-3号位）
  → 空槽点击 → 弹出家园猫咪选择列表
  → 选择猫咪 → 校验不和规则（黑猫/三花猫）
      ├── 通过 → 猫咪上场，技能激活
      └── 冲突 → 提示「这两只猫性格不合，不能同时出战」
  → 点击已上阵猫咪 → 「撤回家园」→ 猫咪回到家园
  → 从家园 Tab 拖拽 → 也可直接上场
```
