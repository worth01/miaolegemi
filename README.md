# 喵了个咪 🐱

一款以猫咪为主题的通勤打卡 + 猫咪养成 App 原型。

## 项目简介

**喵了个咪**将日常通勤打卡与猫咪收集养成相结合，通过消消乐玩法解锁猫咪图鉴，让每天的通勤充满乐趣。

## 功能模块

### 主页大厅
- 每日打卡系统（连续打卡奖励）
- 猫咪轮播展示
- 每日任务
- 小鱼干金币系统
- 暗黑/明亮主题切换

### 猫咪消消乐
- 多层叠放的消消乐玩法
- 3 个关卡（教学关 / 核心关 / 敬请期待）
- 3 种道具：移出、撤销、洗牌
- 临时存放层机制
- 750×1334 全屏适配

### 猫咪抽卡
- 加权概率抽卡（N 75% / R 18% / SR 5% / SSR 2%）
- 80 抽保底 SSR
- 抽卡动画
- 猫咪图鉴（17 种猫咪）

### 个人中心
- 随机昵称生成（猫名+月份+心情）
- 称号系统（佩戴/切换）
- 数据统计（抽卡数/SSR出货/欧气值/图鉴率）
- 卡牌展示区
- 成就徽章

## 猫咪图鉴（17 种）

| 稀有度 | 猫咪 |
|--------|------|
| N | 橘猫、黑猫、白猫、蓝猫、猫爬架 |
| R | 三花猫、布偶猫、暹罗猫 |
| SR | 猫爪印、猫罐头、毛线球、猫薄荷、折耳猫、无毛猫、波斯猫 |
| SSR | 幸运猫、星空猫 |

## 技术栈

- React 18.3.1 + Babel Standalone 7.29.0
- 纯 HTML 单文件，内联 CSS + JSX
- SVG 手绘猫咪插画
- Noto Sans SC 字体
- 双击 HTML 文件即可运行，无需服务器

## 快速开始

```bash
# 克隆仓库
git clone https://github.com/worth01/miaolegemi.git
cd miaolegemi

# 直接双击打开
open 主页大厅.html    # 打开主页大厅
open 猫咪消消乐.html   # 打开消消乐游戏
```
