## 喵了个咪项目简介
一款以猫咪为主题的消除游戏 + 猫咪抽卡 App 原型。
将日常通勤打卡与猫咪收集养成相结合，通过消消乐玩法解锁猫咪图鉴，让每天的通勤充满乐趣。

## 一、游戏定位

### 核心逻辑

```
猫是游戏引擎，不是收集终点。
消消乐是每日互动方式，不是主线。
```

玩家每天的循环：
```
打消消乐 → 获得鱼干 → 决定鱼干用途 → 管理猫咪阵容
```

### 设计原则

- **零压力回归**：长时间离线后回来，养成区的猫依然在等你
- **决策有重量但不焦虑**：鱼干不够时需要做取舍，但不会丢失核心资产
- **变化来自猫咪阵容**：不需要大量关卡，不同猫组合产生不同体验
- **猫咪有双重价值**：编号唯一性 + 技能实用性，缺一不可

## 二、猫咪系统

### 2.1 猫咪种类（12种）

| 稀有度 | 猫咪 | 数量 |
|--------|------|------|
| N | 橘猫、黑猫、白猫、蓝猫 | 4 |
| R | 布偶猫、三花猫、暹罗猫 | 3 |
| SR | 幸运猫、折耳猫、无毛猫、波斯猫 | 4 |

每只猫有两个维度：
- **主动技能**：放在出战席 1-3 位时触发，对应消消乐底部按钮
- **被动技能**：放在观赛区 4-6 位时触发，全局生效

### 2.2 猫咪技能完整表

| 猫咪 | 稀有度 | 主动技能（1-3位） | 被动技能（4-6位） |
|------|--------|-----------------|-----------------|
| 橘猫 | N | **挑出来** · 将一张卡从上方直接移入收集槽 | **吃得多** · 每局计时从开局10秒后计算 |
| 黑猫 | N | **撤回** · 撤销上一步操作 | **灵光** · 收集槽上限+2格 |
| 白猫 | N | **吹毛** · 翻开最顶层一张卡查看下方 | **醒目** · 每局内高亮标记1组可消除组合 |
| 蓝猫 | N | **冻结** · 5秒内计时暂停 | **猫抓板** · 每局自动移出1张干扰卡 |
| 布偶猫 | R | **粘住** · 收集槽已满时可强行塞入1张 | **贵宾** · 鱼干+30% |
| 三花猫 | R | **蹦跶** · 随机跳走2张干扰卡 | **三色守护** · 失败后可重来1次 |
| 暹罗猫 | R | **瞬移** · 将收集槽最底部1张卡移回棋盘 | **吃得快** · 每局计时从开局5秒后计算 |
| 幸运猫 | SR | **洗牌** · 重新打乱当前所有可点击卡牌 | **福星** · 鱼干+50% |
| 折耳猫 | SR | **听声** · 高亮标记2组可消除组合 | **猫之眼** · 棋盘边缘卡牌数量+1 |
| 无毛猫 | SR | **灼烧** · 清除最顶层3张卡牌 | **不屈** · 鱼干+20% |
| 波斯猫 | SR | **吐司** · 将收集槽3张卡打乱放回棋盘 | **优雅步调** · 每局额外1次重来机会 |

猫咪技能显示在卡面，是技能维度，以橘猫举例：显示时：主动技能：**挑出来** · 将一张卡从上方直接移入收集槽 被动技能：**吃得多** · 每局计时从开局10秒后计算


### 2.3 猫咪性格

每只猫抽卡时随机继承 2 个性格词（终身不变）：

```
傲娇、吃货、胆小、黏人、高冷、话痨、懒散、胆大、独行、黏糊、
```

性格词显示在卡面，是社交炫耀维度。

### 2.4 序列号系统

- 每只猫有全局唯一编号（例如 `橘猫 · #00847`）
- 发行时分三个批次：
  - **创世区**（#001-100）：永久特殊标记，通过活动释放
  - **早期区**（#101-500）：首月进入抽卡池
  - **标准区**（#501+）：随玩家增长自然增发
- 初次发现者永久记录在卡面上

## 三、猫咪双区机制

### 3.1 包裹区

- 抽卡后猫咪默认进入包裹区
- **30天倒计时**，不可延长
- 到期后猫咪自动流浪，序列号进入60天冷却 → 回到池子
- 容量上限 20 格，可用鱼干扩容

### 3.2 养成区

- 从包裹区手动领养进入，触发领养仪式动画
- **永久安全**：猫咪在这里不饿、不死、不会流浪，卡牌中不要显示剩余天数，因为在养成区是不会流浪的
- 容量 3-6 格（通过里程碑解锁）
- 前3格 = 主动技能位，后3格 = 被动技能位
- 点击猫咪可拖动排序，决定技能位

### 3.3 养成区 → 流浪（不可逆）

猫从养成区离开只能去流浪，不能回到包裹区。但有一个安全网：

```
点击"送走" → 告别仪式（24小时打包状态）
  → 猫咪在养成区显示背着小背包
  → 24小时内：
    点击"还是留下你" → 花费鱼干 → 猫咪卸下背包回来
  → 24小时后：
    猫咪真正流浪 → 不可逆 → 序列号回到池子
```

### 3.4 双区完整状态流

```
抽卡 → 包裹区（30天）
              ├── 领养 → 养成区（永久安全）
              │           └── 送走 → 24h告别（鱼干可挽回）→ 流浪 → 回池子
              └── 30天到期 → 自动流浪 → 60天冷却 → 回池子
```

## 四、消消乐系统

### 4.1 棋盘卡牌

消消乐棋盘使用 9 种猫咪相关物品卡牌（非猫咪本身）：

```
猫爪印、猫薄荷、毛线球、猫罐头、猫爬架、逗猫棒、猫铃铛、猫条、蝴蝶结
```

- 这些卡牌只在消消乐棋盘出现
- 第一关固定猫爪印，猫罐头，逗猫棒三种
- 第二关9种类型
- 不在抽卡区，不能被收集
- 每日棋盘全服同一张（算法生成 seed）

### 4.2 出战阵容

消消乐底部 3 个按钮 = 养成区前 3 位猫咪的主动技能
之前功能区“移出”功能，给了橘猫，撤销功能给了黑猫，洗牌功能给了幸运猫。
之前功能区，现在对应为[1号位] [2号位] [3号位]  →  消消乐底部3个主动技能按钮
例如：上阵的是橘猫、黑猫、幸运猫对应主动技能按钮为“移出”，“撤销”，“洗牌”功能
```
养成区槽位：
  [1号位] [2号位] [3号位]  →  消消乐底部3个主动技能按钮
  [4号位] [5号位] [6号位]  →  被动加成（全局生效）
```

不同阵容 = 不同策略：
```
追求速度通关 → 橘猫主动+无毛猫被动+黑猫被动
追求鱼干收益 → 布偶猫被动+幸运猫被动+无毛猫被动
追求容错率   → 三花猫被动+波斯猫被动+黑猫主动
追求掌控感   → 折耳猫主动+暹罗猫主动+白猫主动
```

### 4.3 双轨收益

```
随便消几下 → 掉落鱼干（养成资源）     ← 休闲玩家
拼命打高分 → 积分上周榜/解锁称号      ← 竞技玩家
```

## 五、经济系统

### 5.1 双货币

| 货币 | 获取 | 用途 |
|------|------|------|
| 鱼干 | 消消乐产出、每日签到、周榜奖励 | 包裹区扩容、消消乐道具、抽卡辅助、挽留送走的猫 |
| 猫铃铛 | 成就里程碑、满勤奖励、周榜Top N | 抽卡（核心门票） |

### 5.2 鱼干用途明细

| 用途 | 消耗 | 意义 |
|------|------|------|
| 包裹区扩容 | 500 / 1200 / 2500 鱼干 | 扩到 20→30→40→50 格 |
| 消消乐道具 | 10 鱼干/局 | 本局多给1次道具 |
| 抽卡辅助 | 10 鱼干/次 | 提高指定稀有度概率 |
| 告别挽留 | 递增1-5鱼干/天 | 送走的猫24h内可追回 |

### 5.3 经济闭环

```
消消乐（免费玩）→ 产出鱼干
                     ↓
           鱼干 → 扩容/道具/抽卡辅助/挽留
                     ↓
成就/满勤 → 产出猫铃铛（稀缺）
                     ↓
           猫铃铛 → 抽卡 → 包裹区 → 领养进养成区
```

### 8.3 核心 API

```
POST /auth/register            注册
POST /auth/login               登录

GET  /cats/nurturing            养成区猫咪列表
GET  /cats/bag                  包裹区猫咪列表

GET  /cats/:id/detail           猫咪详情
GET  /cats/:serialId/lineage    序列号血统记录

POST /gacha/pull                抽卡（猫铃铛）
GET  /gacha/pity                保底进度

POST /game/start                开始消消乐
POST /game/end                  提交本局结果
GET  /game/leaderboard/today    今日榜单

GET  /user/milestones           里程碑进度
GET  /user/stats                个人统计
```

### 8.4 关键技术决策

**鱼干用账本，不用余额字段**
防止篡改，方便回溯。

**序列号分配用数据库行锁**
```sql
SELECT * FROM cat_serial_registry
WHERE species_id = ? AND status = 'available'
ORDER BY serial_number ASC
LIMIT 1
FOR UPDATE SKIP LOCKED
```

## 十、核心设计理念

> **消消乐上的卡牌是任务，养成区的猫是工具。**
> **包裹区是限时决策，养成区是永久安全。**
> **鱼干不续命，但能挽留。**

不是在做一个消消乐加上收藏系统，而是在做一个**猫咪阵容养成 × 消除游戏**，消消乐是展示猫咪价值的舞台，猫咪是驱动消消乐策略的核心。
