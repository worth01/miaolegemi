## 🗺️ 关卡设计

### 30关 × 6种异形棋盘

```
标准矩形    爱心形❤️    蝴蝶形🦋
菱形💎      沙漏形⏳    波斯双域🔱
```
每种棋盘有独特的点位布局，波斯双域更设有左右独立分区，跨区无法消除，策略深度完全不同。

### 6种干扰牌机制

| 干扰牌 | 出现关卡 | 效果 |
|--------|---------|------|
| 舔毛牌 | 第2关 | 进槽后永久占位，无法消除 |
| 冰冻牌 | 第3关 | 冻结收集槽第1格全程无法使用 |
| 粘连牌 | 第5关 | 点击时连带相邻1张一同入槽 |
| 三花牌 | 第6关 | 每3秒随机变换图案 |
| 小狗牌 | 第8关 | 一旦入槽立即判定本局失败 ⚠️ |
| 真身牌 | 第9关 | 伪装成普通牌，点击后才露出真身 |

### 羁绊系统

特定猫咪组合同时上阵，触发技能升级：
**🐾 好伙伴羁绊**（橘猫 + 布偶猫同时出战）
- 橘猫「挑出来」升级为**升级版挑出来**
- 从棋盘上指定1张牌，连同剩余2张同类型牌，三张直接消除，不经过收集槽
**🐾 不和羁绊**（三花猫 + 黑猫不能同时出战）
三花猫与黑猫不和，不能同时上场
羁绊系统等待挖掘

## 💰 经济系统

### 鱼干产出（完全透明）

```
每6次手动有效三消 = 1个基础鱼干
每局基础鱼干上限：3个
额外鱼干 = 基础鱼干 × 猫咪被动加成（向下取整）
额外鱼干上限：4个（狡黠性格可解锁）
单局总鱼干上限：7个
注册即得100鱼干
```

### 抽卡保底机制

| 保底类型 | 触发条件 | 保证内容 |
|---------|---------|---------|
| SR小保底 | 每累计20抽 | 至少SR（30%概率升为SSR） |
| SSR大保底 | 每累计80抽 | 必定SSR |
注册即送，SSR 幸运猫。

## 🚀 快速开始

### 直接试玩（推荐）

**[👉 点击这里直接玩](https://worth01.github.io/miaolegemi/)**

手机、电脑、平板均可，无需安装任何东西。

### 本地运行

```bash
# 克隆仓库
git clone https://github.com/worth01/miaolegemi.git
cd 仓库名

# 前端：直接用浏览器打开
open index.html
# 或者用 VS Code Live Server 插件打开

# 后端（可选，需要完整养成系统）
cd server
cp .env.example .env    # 填入你的数据库配置
npm install
npm run dev
```

### 环境要求

- 前端：任意现代浏览器（Chrome / Safari / Firefox）
- 后端：Node.js 18+ · PostgreSQL 14+
